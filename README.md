# Online Reservation System

## 📌 Objective

The objective of this project is to develop a console-based Online Reservation System using Java. The application allows users to log in, reserve railway tickets, generate a PNR number, view reservation details, and cancel reservations. Reservation records are stored using file handling.

---

## ✨ Features

* User Login Authentication
* Railway Ticket Reservation
* Automatic PNR Number Generation
* Train Name Lookup using Train Number
* Ticket Cancellation using PNR
* File Handling for Data Storage
* Display All Reservations
* Console-based Interactive Menu

---

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Scanner Class
* FileWriter
* CSV Files

## ▶️ How to Run

1. Open the project in your Java IDE or Command Prompt.
2. Navigate to the `src` folder.
3. Compile the Java files:

```bash
javac Main.java FileManager.java User.java PNRGenerator.java
```

4. Run the application:

```bash
java Main
```
